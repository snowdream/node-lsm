# node-nlsm
Linux Script Manager with Node and commander.js

## Install
```
npm install nlsm -g
```

## Usage
```
nlsm install **
nlsm uninstall **
```

## Supported Scripts:
1. docker: https://get.docker.com/
1. ss,ssr,shadowsocks: https://github.com/teddysun/shadowsocks_install
 https://teddysun.com/486.html
